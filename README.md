# prompt-create-video

คลังเก็บ prompt สำหรับสร้างวิดีโออวยพรองค์เทพ (Sri Kanesh Devalaya) ด้วย **Grok Imagine Video**

แต่ละไฟล์คือชุด prompt ที่ผ่านการวิเคราะห์ภาพองค์เทพแล้ว พร้อมบทพูดภาษาไทยโบราณศักดิ์สิทธิ์ (ข้า/เจ้า) ที่ฝังคีย์เวิร์ดแบรนด์ "สถิตอยู่ที่ สีคะเนตร เทวาลัย" ไว้เสมอ — สร้างโดยสกิล `sri-kanesh-deity-video-prompt-generator`

## โครงสร้างโฟลเดอร์

```
prompt-create-video/
├── README.md
├── prompts/
│   ├── _template.md          ← แม่แบบเปล่าสำหรับสร้างองค์ใหม่
│   └── lakshmi/
│       └── 2026-07-21-lakshmi-lotus-elephants.md
```

- แยกโฟลเดอร์ตามองค์เทพ (`prompts/<ชื่อองค์>/`)
- ตั้งชื่อไฟล์แบบ `YYYY-MM-DD-<ชื่อองค์>-<จุดสังเกตของภาพ>.md` เพื่อให้ไล่ตามลำดับเวลาและแยกภาพเดิม/ภาพใหม่ของแต่ละองค์ได้ง่าย

## วิธีใช้

1. เปิดไฟล์ prompt ของภาพที่ต้องการ
2. คัดลอกแต่ละ "ชุด" ไปวางใน Grok Imagine Video ตามลำดับ (ชุดที่ 1 = image-to-video จากภาพต้นฉบับ, ชุดถัดไป = video-extend จากเฟรมสุดท้าย)
3. ตั้งค่า resolution ในหน้าเว็บ Grok เอง (อย่าใส่คำขอ resolution ในข้อความ prompt)

## หมายเหตุ

- Lip-sync จากภาพนิ่งอาจไม่เนียนสม่ำเสมอทุกครั้ง โดยเฉพาะช็อตโคลสอัพหน้า — อาจต้องรันซ้ำบางชุด
- Extend ต่อกันไม่ควรเกิน 2-3 ครั้งติดกัน ก่อนคุณภาพจะเริ่มลด
