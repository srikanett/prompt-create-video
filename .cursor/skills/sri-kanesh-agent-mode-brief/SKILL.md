# Sri Kanesh Agent Mode Brief Generator

เขียน **brief เดียว** สำหรับ Grok Imagine Agent Mode: แนบภาพองค์เทพ 1 ภาพ + วาง brief 1 ครั้ง → agent วางแผนและสร้างวิดีโอให้ **3 คลิปแยกกัน** คลิปละ 10-15 วินาที ในรอบเดียว

## ความรู้พื้นฐานเรื่อง Agent Mode (สำคัญ ต้องเขียน brief ให้สู้กับพฤติกรรม default)

- Agent Mode ชอบสร้างคลิปฐาน 6 วินาทีแล้ว "ตัดต่อรวม" เป็นหนังเรื่องเดียว → brief ต้องสั่งชัดว่า **ต้องการ 3 คลิปแยกกัน (separate deliverables) ไม่ใช่หนังเรื่องเดียวตัดต่อรวม** และแต่ละคลิปยาว 10-15 วิ (ให้ agent ใช้ image-to-video ที่รองรับสูงสุด 15 วิ หรือ extend ต่อจนถึงความยาวเป้าหมาย)
- ทุกคลิปเริ่มจาก **ภาพเดียวกันที่แนบ** เพื่อคุมหน้าตาองค์เทพให้เหมือนกันทั้ง 3 คลิป
- ระบุเสียงพากย์ + บทพูดในแต่ละคลิปให้ชัดใน brief (ขอ native audio จากโมเดลวิดีโอ 1.5) — ถ้ารอบไหนออกมาไม่มีเสียง ให้กดรีรันเฉพาะ node นั้นแล้วย้ำเรื่อง AUDIO ใน node prompt

## กฎเหล็ก (เหมือน skill วิดีโอหลักทุกข้อ)

1. คลิปแรกต้องมี "ข้า[ชื่อองค์] สถิตอยู่ที่ สีคะเนตร เทวาลัย" เสมอ
2. บทพูด 3 คลิปเป็นภาษาไทยโบราณ (ข้า/เจ้า) ต่อเนื่องกันเป็นเรื่องเดียว: คลิป 1 = เรียกให้ขอพร+แนะนำตัว, คลิป 2 = ปัดเป่าทุกข์, คลิป 3 = อวยพรปิดท้าย+เชิญมากราบไหว้
3. เสียงเดียวกันทั้ง 3 คลิป: หญิงไทยวัยราว 40 น้ำเสียงตามโทนองค์ (สงบ-เมตตา หรือ ขรึมทรงอำนาจ)
4. ของประจำองค์ต้องตรงองค์ (ดูตารางใน skill sri-kanesh-deity-video ถ้ามี หรือ: พิฆเนศ=งวง/ประทานพร, ลักษมี=เหรียญทอง/บัว, อุมา=สงบแบบแม่, ทุรคา=หลายกร/อาวุธ/สิงโต, กาลี=พวงกะโหลก/อาวุธ/เกรี้ยวกราด)
5. หนึ่งคลิป หนึ่ง action หลัก, บทพูดไม่เกิน ~50 คำ/คลิป

## Template Brief (แทนค่า [ ] แล้ววางใน Agent Mode พร้อมแนบภาพ)

```
สร้างวิดีโอ 3 คลิปแยกกัน (3 separate video files ไม่ใช่หนังเรื่องเดียวตัดต่อรวม) จากภาพ [ชื่อองค์] ที่แนบมา ทุกคลิปใช้ภาพนี้เป็นเฟรมแรกหรืออ้างอิงหน้าตาเดียวกัน คลิปละ 10-15 วินาที แนวตั้ง 9:16 720p ใช้โมเดลวิดีโอที่มี native audio

ทุกคลิป: เสียงพากย์หญิงไทยวัยราว 40 น้ำเสียง[ขรึมทรงอำนาจ/อ่อนโยนเมตตา] พูดภาษาไทยชัดเจน lip-sync ตรงคำพูด แสง[แบ็คไลท์จัดจ้านดราม่า/ทองนวลอบอุ่น] คุมหน้าตา เครื่องทรง และแสงให้เหมือนกันทั้ง 3 คลิป

คลิปที่ 1 — [action เช่น สีหน้าเปลี่ยนเป็นเกรี้ยวกราด อาวุธแกว่งช้าๆ จ้องกล้อง]
พูด: "ใครผ่านมาเห็นข้าในวันนี้ เจ้าจงรีบตั้งจิตอธิษฐานขอพรให้ไว ข้า[ชื่อองค์] สถิตอยู่ที่ สีคะเนตร เทวาลัย จะอวยพรให้ลูกทุกคน[พรตามองค์ เช่น หลุดพ้นจากหนี้สินและทุกข์กรรมทั้งปวง]"

คลิปที่ 2 — [action ต่อเนื่อง เช่น ก้าวเดินเข้าหากล้องอย่างมั่นคง]
พูด: "หากเจ้ากำลังพบเจอ[ความทุกข์/ความฝืดเคือง] ขอให้สิ่งนั้นจงหลุดพ้นจากตัวเจ้าอย่างฉับพลัน ข้าจะบันดาล[ความสุข/โชคลาภ]ให้เจ้าทุกคน"

คลิปที่ 3 — [action ปิดท้าย เช่น ยกพระหัตถ์ประทานพร แสงเปล่งสว่างขึ้น]
พูด: "ชีวิตเจ้าจงมีแต่ความสมหวัง ดั่งปรารถนา จงมากราบไหว้ข้า ณ สีคะเนตร เทวาลัย แล้วพรอันประเสริฐจะบังเกิดแก่เจ้า"

เสียงพื้นหลังทุกคลิป: [เสียงหึ่งดราม่าเบาๆ ลมพัดต่ำ / เสียงกระดิ่งแว่ว ลมพัดเบา] ต่อเนื่องโทนเดียวกัน
```

## ขั้นตอนใช้งานใน Grok

1. เปิด grok.com → Imagine → เปิดสวิตช์ Agent Mode (มุมช่องพิมพ์ ต้องเป็นบัญชีแบบเสียเงิน ใช้ได้เฉพาะเว็บ)
2. ลากภาพองค์เทพลง canvas → วาง brief ที่แทนค่าแล้ว → รัน
3. agent จะวางแผนเป็น node tree — ตรวจ preview แต่ละ node ได้ ถ้าคลิปไหนไม่ตรง (เสียงหาย/หน้าเพี้ยน/สั้นไป) คลิก node นั้นแล้ว reprompt เฉพาะจุด ไม่ต้องรันใหม่ทั้งหมด
4. Export ทีละคลิป (3 ไฟล์แยก)

## ข้อควรระวัง

- Agent Mode ยังเป็น beta พฤติกรรมเปลี่ยนได้ — ถ้า agent ดื้อรวมเป็นหนังเรื่องเดียว ให้ reprompt ย้ำว่า "3 separate exports"
- ถ้าคลิปออกมา 6 วิแทนที่จะเป็น 10-15 วิ ให้สั่ง extend node นั้นต่อ ("extend this clip to 12 seconds, continue the same motion")
- หน้าตาองค์เทพอาจ drift ข้ามคลิป — ถ้าเพี้ยนให้ลากภาพต้นฉบับใส่ node นั้นซ้ำเป็น reference
- ไม่แน่ใจว่าองค์ไหน → ถามผู้ใช้ก่อน ห้ามเดา
