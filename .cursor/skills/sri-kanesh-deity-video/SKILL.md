# Sri Kanesh Deity Video Generator

Pipeline: **วิเคราะห์ภาพอย่างชาญฉลาด → (text-to-image ถ้าไม่มีภาพ) → image-to-video → video-extend → video-extend** สร้างวิดีโออวยพร 3 ชุดต่อเนื่องเป็นเรื่องเดียว

## ขั้นที่ 1: วิเคราะห์ภาพก่อนเสมอ (สำคัญที่สุด)

ก่อนเขียน prompt วิดีโอ ต้องวิเคราะห์ภาพให้ครบ 5 ด้าน:

**1.1 ตัวละคร** — เป็นองค์ไหน/ใครบ้าง กี่ตัวละคร เพศใด (ดูจากลักษณะในภาพ+ความรู้เรื่ององค์เทพ เช่น พิฆเนศ=บุรุษ, พระแม่ทุกองค์=สตรี) ถ้าไม่แน่ใจว่าองค์ไหนให้ถามผู้ใช้ ห้ามเดา

**1.2 ท่าทางในภาพ** — นั่ง ยืน เดิน นอน ประทับบนบัลลังก์/พาหนะ ฯลฯ แล้วเลือก motion ที่ต่อยอดจากท่านั้นอย่างสมเหตุสมผล (อิสระเต็มที่ ไม่มีกฎตายตัว) ตัวอย่างแนวคิด:
- นั่งสมาธิ/ประทับ → ขยับเบาๆ พูดตรงกล้อง กล้อง close-up เข้าช้าๆ เรื่อยๆ
- ยืน → ก้าวเดินเข้าหากล้องพร้อมพูด (เช่น กาลียืน → เดินหน้าเกรี้ยวกราด อาวุธแกว่ง)
- ถืออาวุธ/สิ่งของ → ให้ของนั้นเคลื่อนไหวประกอบ (อาวุธแกว่ง เหรียญไหล ดอกไม้โปรย)
- นอน/เอนกาย → ลืมตา ยกพระหัตถ์ประทานพร กล้องแพนช้า

**1.3 บรรยากาศรอบตัว** — ทุกอย่างในฉากต้องมีชีวิต ไม่ใช่ขยับแค่ตัวละคร: เปลวไฟไหว ควันธูปลอย ผ้าพลิ้ว น้ำไหล ใบไม้ไหว เมฆเคลื่อน แสงกะพริบ ฝุ่นทองลอย — เลือกตามสิ่งที่มีจริงในภาพ

**1.4 Soundscape** — เสียงประกอบต้องตรงกับสิ่งที่เห็น+โทนอารมณ์: เสียงธรรมชาติ (นก น้ำตก ลม ฝน) เสียงศาสนสถาน (ระฆัง กระดิ่ง สวด) เสียงดุดัน (อสูรคำราม ฟ้าร้อง อาวุธหวีดผ่านอากาศ เปลวไฟลุก) — ระบุเป็นรายการชัดๆ ใน prompt

**1.5 เพลงบรรเลง background** — ใส่ทุกวิดีโอ เบาๆ ไม่กลบเสียงพูด เลือกแนวตามบรรยากาศภาพ (เช่น ฉากสงบ=เครื่องสายอินเดียแผ่ว, ฉากดุ=กลองต่ำจังหวะช้า+โดรนเสียงลึก) เขียนกำกับว่า "เพลงบรรเลงเบาๆ อยู่เบื้องหลัง เสียงพูดเด่นชัดกว่าเสมอ"

## ขั้นที่ 2: กำหนดเสียงพากย์

- **วิเคราะห์เพศ+วัยจากภาพ** แล้วกำหนดเสียงตามความเหมาะสมของตัวละครนั้น — องค์เทพบุรุษ=เสียงชาย องค์เทพสตรี=เสียงหญิง มนุษย์=ตามที่เห็นในภาพ
- **ห้ามใช้เสียงที่ฟังดูอายุต่ำกว่า 25 ปี** ทุกตัวละคร
- **หลายตัวละคร = คนละเสียงเสมอ** ระบุ descriptor แยกชัดใน prompt เช่น "เสียงชายไทยวัยกลางคน ทุ้มลึกทรงอำนาจ (องค์เทพ)" กับ "เสียงหญิงไทยวัยราว 30 นอบน้อม (ผู้ขอพร)"
- descriptor เดียวกันต้องคงที่ทุกชุดของวิดีโอเดียวกัน (ชุดถัดไปเขียน "เสียงเดิม")

## กฎเหล็ก

1. ต้องมีท่อน **"ข้า[ชื่อองค์] สถิตอยู่ที่ สีคะเนตร เทวาลัย"** ในท่อนแรกของเสียงองค์เทพเสมอ
2. ของประจำองค์ต้องตรงองค์: พิฆเนศ=งวง/ขวาน/โมทกะ, ลักษมี=บัว/เหรียญทอง, อุมา=สงบแบบแม่, ทุรคา=หลายกรอาวุธ/สิงโต (ดุแบบปกป้อง), กาลี=พวงกะโหลก/ลิ้นแดง/อาวุธ (เกรี้ยวกราด) — ห้ามสลับ
3. **สร้าง 3 ชุดเสมอ** (เว้นแต่ผู้ใช้ระบุจำนวนอื่น): ชุดที่ 1=image-to-video จากภาพ / ชุดที่ 2 และ 3=video-extend ต่อจากเฟรมสุดท้ายของชุดก่อนหน้าเป็นลูกโซ่ (1→2→3 ไม่สร้างใหม่จากภาพซ้ำ)
4. หนึ่งชุดหนึ่ง action หลัก, บทพูดรวมทุกตัวละครไม่เกิน ~50 คำ/ชุด, บทพูดและ motion ทั้ง 3 ชุดต้องต่อเนื่องเป็นเรื่องเดียว: **ชุด 1**=เรียกให้ขอพร+แนะนำตัวพร้อมคีย์เวิร์ดแบรนด์ / **ชุด 2**=ปัดเป่าทุกข์-ประทานพรหลัก / **ชุด 3**=อวยพรปิดท้าย+เชิญมากราบไหว้ ณ สีคะเนตร เทวาลัย

## กรณี 2 ตัวละคร (เทพ+มนุษย์ หรือ เทพ+เทพ)

เขียนเป็นบทสนทนาโต้ตอบ: มนุษย์ขอพร → องค์เทพประทานพร (หรือเทพ 2 องค์อวยพรเสริมกัน) แบ่งจังหวะข้ามชุดได้ เช่น ชุด 1 มนุษย์พนมมือขอพร+เทพแนะนำตัวพร้อมคีย์เวิร์ดแบรนด์, ชุด 2 เทพประทานพรเต็มท่อน กล้องสลับ close-up ผู้พูดแต่ละฝั่งหรือเห็นทั้งคู่ตามความเหมาะสมของภาพ

## รูปแบบ Prompt แต่ละชุด

```
[คำบรรยาย motion ตัวละคร + บรรยากาศรอบตัวที่เคลื่อนไหว + มุมกล้อง]. AUDIO: [voice descriptor ตัวละคร A]: "[บทพูด]" [ถ้ามีตัวละคร B: voice descriptor B]: "[บทพูด]". เสียงประกอบ: [soundscape ตามภาพ]. เพลงบรรเลงเบาๆ เบื้องหลัง: [แนวตามบรรยากาศ] เสียงพูดเด่นกว่าเพลงเสมอ
```

## ตัวอย่าง (กาลี — ภาพยืนถืออาวุธ)

```
ชุดที่ 1 (image-to-video):
พระแม่กาลีก้าวเดินเข้าหากล้องช้าๆ ใบหน้าเกรี้ยวกราด อาวุธในหลายกรแกว่งไปมา เปลวไฟด้านหลังลุกไหว ควันลอยอ้อยอิ่ง ผ้าทรงพลิ้วตามแรงลม กล้องถอยรับพร้อมเก็บเต็มองค์
AUDIO: เสียงหญิงไทยวัยกลางคน ขรึมทรงอำนาจ: "ใครผ่านมาเห็นข้าในวันนี้ เจ้าจงรีบตั้งจิตอธิษฐานขอพรให้ไว ข้าพระแม่กาลี สถิตอยู่ที่ สีคะเนตร เทวาลัย จะอวยพรให้ลูกทุกคนหลุดพ้นจากหนี้สินและทุกข์กรรมทั้งปวง"
เสียงประกอบ: เปลวไฟลุกโชน อาวุธหวีดผ่านอากาศเบาๆ ฟ้าครางไกลๆ
เพลงบรรเลงเบาๆ เบื้องหลัง: กลองต่ำจังหวะช้า+โดรนเสียงลึก เสียงพูดเด่นกว่าเพลงเสมอ

ชุดที่ 2 (video-extend จากเฟรมสุดท้ายของชุดที่ 1):
พระแม่กาลีหยุดยืนเต็มเฟรม ยกพระหัตถ์ข้างหนึ่งขึ้นประทานพร แสงเปล่งสว่างขึ้น เปลวไฟและควันยังไหวต่อเนื่อง กล้อง close-up เข้าใบหน้าช้าๆ
AUDIO: เสียงเดิม: "หากเจ้ากำลังพบเจอความทุกข์ ขอให้ความทุกข์นั้นหลุดพ้นจากตัวเจ้าอย่างฉับพลัน ข้าจะบันดาลความสุขให้เจ้าทุกคน"
เสียงประกอบ: เปลวไฟเบาลง เสียงกระดิ่งแว่วขึ้น
เพลงบรรเลงเบาๆ เบื้องหลัง: ต่อเนื่องจากชุดก่อน

ชุดที่ 3 (video-extend จากเฟรมสุดท้ายของชุดที่ 2):
พระแม่กาลีลดพระหัตถ์ลงช้าๆ สีหน้าผ่อนจากเกรี้ยวกราดเป็นสงบทรงเมตตา แสงทองอบอุ่นแผ่รอบพระวรกาย ควันธูปลอยขึ้นเบื้องหน้า กล้องถอยออกช้าๆ เก็บเต็มองค์อีกครั้ง
AUDIO: เสียงเดิม: "ชีวิตเจ้าจงมีแต่ความสมหวัง ดั่งปรารถนา จงมากราบไหว้ข้า ณ สีคะเนตร เทวาลัย แล้วพรอันประเสริฐจะบังเกิดแก่เจ้า"
เสียงประกอบ: เสียงกระดิ่งชัดขึ้น เสียงสวดแผ่วไกลๆ
เพลงบรรเลงเบาๆ เบื้องหลัง: ต่อเนื่องจากชุดก่อน ผ่อนลงจบท้ายคลิป
```

## Text-to-Image (เมื่อไม่มีภาพ)

`ภาพ [ชื่อองค์] สไตล์ hyper-realistic digital sculpture โทนสีม่วงเข้ม-ดำ ทองคำ แดงทับทิม [ท่าทาง+ของประจำองค์] [แสง: สงบ=ทองนวลอบอุ่น / ดุ=แบ็คไลท์จัดจ้านดราม่า] พื้นหลังมีองค์ประกอบที่เคลื่อนไหวได้ในวิดีโอ (เปลวไฟ/ควันธูป/ม่านผ้า) แนวตั้ง คมชัดสูง` แล้วต่อขั้นที่ 1 ตามปกติ

## ข้อควรระวัง
- Lip-sync จากภาพนิ่งอาจไม่เนียนทุกครั้ง เตรียมรันซ้ำ / 2 ตัวละครพูด ยิ่งต้องเช็คว่าเสียงตรงปากตรงตัว
- สูงสุด 15 วิ/ชุด, extend ต่อลูกโซ่ 2 ครั้ง (รวม 3 ชุด) คือจุดสมดุลพอดี — เกินกว่านี้คุณภาพและหน้าตาตัวละครจะเริ่ม drift ถ้าชุดที่ 3 หน้าเพี้ยนให้รันชุดนั้นซ้ำ
- อย่ายัดหลาย action ใน prompt เดียว บรรยากาศเคลื่อนไหวได้หลายอย่างแต่ตัวละครทำทีละอย่าง
